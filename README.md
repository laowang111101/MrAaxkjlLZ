# 银行管理系统 python87

本项目是基于Python的银行管理系统，使用Flask作为Web开发框架，MySql用于存储和管理数据。

## 技术栈

- Python 3.11
- MySql 8.0
- Flask 3.0.0

## 功能模块

### 客户管理

- 创建客户
- 删除客户
- 查询单个客户
- 查询所有客户
- 修改客户信息

### 账户管理

- 开卡
- 开支票
- 删除账户
- 修改账户
- 查询单个账户
- 查询所有账户

### 贷款管理

- 发放贷款
- 删除贷款
- 查询单笔贷款
- 查询所有贷款
- 支付贷款

### 业务统计

- 按业务分类统计
- 按月份统计
- 按季度统计
- 按年份统计

## 系统角色

系统主要涉及以下角色：

- 客户
- 银行柜员
- 管理员

## 运行环境

- 操作系统：支持Linux、Windows等
- Python环境：Python 3.11及以上版本
- 数据库：MySql 8.0

## 部署步骤

1. 安装Python 3.11
2. 安装MySql 8.0
3. 安装Flask 3.0.0
4. 克隆项目代码
5. 导入数据库结构
6. 运行项目
